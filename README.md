# Waste2Worth

An intelligent bagasse quality classification and utilization recommendation system developed using Machine Learning and Image Processing.

## Technologies Used

- Python
- Streamlit
- Random Forest
- MySQL
- PIL
- NumPy
- Scikit-learn

## Features

- User Login & Signup
- Bagasse Image Upload
- Image Analysis
- Quality Prediction
- Explainable AI
- Utilization Recommendation
- Prediction History
